# 🏗️ TowerClonexx

A 3D precision block-stacking game built with Unity. The core objective is to drop blocks at the exact right moment to build the highest tower possible, maintaining streaks to maximize the score.

## 🎮 Controls

* **Spacebar** Drop the block.
* **ESC:** Pause Game.
